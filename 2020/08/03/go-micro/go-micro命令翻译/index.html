    go-micro命令(中文翻译)

    项目名字:
    micro - A microservice runtime

    使用命令:
    micro [global options] command [command options] [arguments…]

    版本:
    v2.9.1

    命令:
    api 运行api网关
    auth auth运行auth服务
    login 使用token登录
    whoami 登录帐户信息
    bot 运行chatops bot
    cli 运行交互式CLI
    call 调用服务，例如微调用greeter Say.Hello’{“ name”：“ John” }
    stream 创建服务流
    publish 将消息发布到topic
    stats 查询服务的统计信息
    env 获取/设置Micro cli环境
    file 在本地计算机和服务器间移动文件
    list 列出注册表或网络中的项目
    register 将一个项目注册到注册表
    deregister 从注册表中注销一个项目
    get 从注册表中获取一个项目
    broker 消息代理运行
    health 检查服务健康状态
    proxy 运行服务代理
    router 执行网络路由
    tunnel 执行网络隧道
    network 执行网络节点
    registry 运行服务注册表
    runtime 运行runtime
    run Required usage: micro run [source]
    kill Require usage: micro kill [source]
    update Require usage: micro update [source]
    services Require usage: micro services
    status Require usage: micro status [service] [version]
    logs 获取一个服务的日志
    debug 运行调试服务
    trace 跟踪服务
    server 运行服务
    service 运行微服务
    store 运行微商店服务
    new 创建服务模板
    plugin 插件命令
    web 运行Web仪表板
    config 管理配置值
    init 初始化操作
    help, h 显示命令列表或一个命令的帮助

    GLOBAL OPTIONS:
    –client value go-micro客户端; rpc [$MICRO_CLIENT]
    –client_request_timeout value 设置客户端请求超时。例如 500ms, 5s, 1m. Default: 5s [$MICRO_CLIENT_REQUEST_TIMEOUT]
    –client_retries value 设置客户端重试次数。 默认值：1(default: 1) [$MICRO_CLIENT_RETRIES]
    –client_pool_size value 设置客户端连接池大小。默认值：1 (default: 0) [$MICRO_CLIENT_POOL_SIZE]
    –client_pool_ttl value 设置客户端连接池ttl。例如500ms, 5s, 1m. 默认值: 1m [$MICRO_CLIENT_POOL_TTL]
    –register_ttl value 以秒为单位注册TTL (default: 60) [$MICRO_REGISTER_TTL]
    –register_interval value 以秒为单位的注册间隔 (default: 30) [$MICRO_REGISTER_INTERVAL]
    –server value go-micro服务端; rpc [$MICRO_SERVER]
    –server_name value 服务名. go.micro.srv.example [$MICRO_SERVER_NAME]
    –server_version value 服务版本. 1.1.0 [$MICRO_SERVER_VERSION]
    –server_id value 服务server_id。如果未指定，则自动生成 [$MICRO_SERVER_ID]
    –server_address value 服务绑定地址。默认：127.0.0.1:8080 [$MICRO_SERVER_ADDRESS]
    –server_advertise value 服务发现注册时，代替server_address使用 127.0.0.1:8080 [$MICRO_SERVER_ADVERTISE]
    –server_metadata value 定义元数据的键值对的列表。 version=1.0.0 [$MICRO_SERVER_METADATA]
    –broker value 代理发布/订阅。可选 http, nats, rabbitmq [$MICRO_BROKER]
    –broker_address value 逗号分隔的代理地址列表 [$MICRO_BROKER_ADDRESS]
    –profile value 用于cpu和内存统计信息的调试分析器 [$MICRO_DEBUG_PROFILE]
    –registry value 用于发现的注册表。可选 etcd, mdns [$MICRO_REGISTRY]
    –registry_address value 逗号分隔的注册表地址列表 [$MICRO_REGISTRY_ADDRESS]
    –runtime value 用于构建和运行服务的runtime。例如 local, kubernetes (default: “local”) [$MICRO_RUNTIME]
    –runtime_source value 用于构建和运行服务的运行时源。例如 github.com/micro/service (default: “github.com/micro/services”) [$MICRO_RUNTIME_SOURCE]
    –selector value 选择器，用于选择要查询的节点 [$MICRO_SELECTOR]
    –store value 用于键值存储的商店 [$MICRO_STORE]
    –store_address value 逗号分隔的商店地址列表 [$MICRO_STORE_ADDRESS]
    –store_database value 基础商店的数据库选项 [$MICRO_STORE_DATABASE]
    –store_table value 基础商店的表选项 [$MICRO_STORE_TABLE]
    –transport value 所使用的传输机制; http [$MICRO_TRANSPORT]
    –transport_address value 以逗号分隔的传输地址列表 [$MICRO_TRANSPORT_ADDRESS]
    –tracer value 用于分布式跟踪的跟踪器，例如 memory, jaeger [$MICRO_TRACER]
    –tracer_address value 以逗号分隔的跟踪器地址列表 [$MICRO_TRACER_ADDRESS]
    –auth value 角色的身份验证基于访问控制，例如 service [$MICRO_AUTH]
    –auth_id value 用于客户端身份验证的帐户ID [$MICRO_AUTH_ID]
    –auth_secret value 用于客户端身份验证的帐户密码 [$MICRO_AUTH_SECRET]
    –auth_namespace value 服务auth帐户的命名空间 (default: “go.micro”) [$MICRO_AUTH_NAMESPACE]
    –auth_public_key value 用于JWT auth的公共密钥（以base64编码的PEM） [$MICRO_AUTH_PUBLIC_KEY]
    –auth_private_key value 用于JWT auth的私钥（以base64编码的PEM） [$MICRO_AUTH_PRIVATE_KEY]
    –auth_provider value 用于登录用户的身份验证提供者 [$MICRO_AUTH_PROVIDER]
    –auth_provider_client_id value 用于oauth的客户端ID [$MICRO_AUTH_PROVIDER_CLIENT_ID]
    –auth_provider_client_secret value 用于oauth的客户端密钥 [$MICRO_AUTH_PROVIDER_CLIENT_SECRET]
    –auth_provider_endpoint value 用于oauth的enpoint [$MICRO_AUTH_PROVIDER_ENDPOINT]
    –auth_provider_redirect value 用于oauth的重定向 [$MICRO_AUTH_PROVIDER_REDIRECT]
    –auth_provider_scope value 用于oauth的范围 [$MICRO_AUTH_PROVIDER_SCOPE]
    –config value 用于获取配置的配置源 [$MICRO_CONFIG]
    –local 仅本地开发，默认是true. (default: false)
    –enable_acme 通过“让我们加密”启用ACME支持。还应指定ACME主机。(default: false) [$MICRO_ENABLE_ACME]
    –acme_hosts value 以逗号分隔的用于管理ACME证书的主机名列表 [$MICRO_ACME_HOSTS]
    –acme_provider value 用于与Let’s Encrypt通信的提供程序。有效选项：autocert, certmagic [$MICRO_ACME_PROVIDER]
    –enable_tls 启用TLS支持。需要指定证书和密钥文件 (default: false) [$MICRO_ENABLE_TLS]
    –tls_cert_file value TLS证书文件的路径 [$MICRO_TLS_CERT_FILE]
    –tls_key_file value TLS密钥文件的路径 [$MICRO_TLS_KEY_FILE]
    –tls_client_ca_file value 用于验证客户端的CA文件的TLS的路径 [$MICRO_TLS_CLIENT_CA_FILE]
    –api_address value 设置api地址。例如 0.0.0.0:8080 [$MICRO_API_ADDRESS]
    –namespace value 设置微服务名称空间 (default: “micro”) [$MICRO_NAMESPACE]
    –proxy_address value 通过指定的HTTP地址进行代理请求 [$MICRO_PROXY_ADDRESS]
    –web_address value 设置Web UI地址，例如 0.0.0.0:8082 [$MICRO_WEB_ADDRESS]
    –network value 设置网络名称： local, go.micro [$MICRO_NETWORK]
    –network_address value 设置微网络地址。例如 :9093 [$MICRO_NETWORK_ADDRESS]
    –router_address value 设置路由地址，例如 :8084 [$MICRO_ROUTER_ADDRESS]
    –gateway_address value 设置微型默认网关地址，例如 :9094 [$MICRO_GATEWAY_ADDRESS]
    –tunnel_address value 设置隧道地址，例如 :8083 [$MICRO_TUNNEL_ADDRESS]
    –api_handler value 指定用于将HTTP请求映射到服务的请求处理程序; {api, proxy, rpc} [$MICRO_API_HANDLER]
    –api_namespace value 设置API使用的名称空间，例如 com.example.api [$MICRO_API_NAMESPACE]
    –web_namespace value 设置Web代理使用的名称空间，例如 com.example.web [$MICRO_WEB_NAMESPACE]
    –web_url value 设置用于Web仪表板的主机，例如 web.example.com [$MICRO_WEB_HOST]
    –enable_stats 启用统计信息 (default: false) [$MICRO_ENABLE_STATS]
    –auto_update 启用自动更新 (default: false) [$MICRO_AUTO_UPDATE]
    –update_url value 设置从以下的URL来检索系统更新 (default: “https://go.micro.mu/update") [$MICRO_UPDATE_URL]
    –report_usage 报告使用情况统计信息 (default: true) [$MICRO_REPORT_USAGE]
    –env value, -e value 覆盖环境 [$MICRO_ENV]
    –plugin value 逗号分隔的插件列表，例如broker/rabbitmq, registry/etcd, micro/basic_auth, /path/to/plugin.so [$MICRO_PLUGIN]
    –help, -h 帮助 (default: false)
    –version 打印版本 (default: false)
